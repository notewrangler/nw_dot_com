spot-ON Merchandising

This project is a work in progress--my first Ruby on Rails application. There are two entry points-- one is for administrators to add new contractors, add and manage jobs, schedule contractors for work, assign team leads, set pay rates for various workers, and keep track of in-progress and finished jobs.

The second entry point is for contractors, who can view available jobs, indicate availability, see upcoming assigned work, submit time reports, and keep track of past assignments.

Features are being added as time permits, to enhance the UX as well as improving funtionality. The latest is a log in feature designed to be used at the worksite, from the agent's phone. On log in, if the agent is assigned for work that particular day, the home page includes a button leading directly to a sign-in view that shows the current time and has a single button for signing in. When the agent signs in, the sign-in time is sent to the server and stored in the database for later retreival by administrators.
